Os aplicativos são responsáveis pelo contato, entrega e gestão de valores dentro da organização. São os principais por
gerar faturamento.

### Pet Finder

Pet finder contempla a facilidade de encontrar pets perdidos ou denuncias em geral.

### Pet Shop

Pet Shop é um módulo integrado dentro do aplicativo Pet Finder. Lá se permite realizar publicações de compra ou venda e
negociações de pets ou produtos específicos para tutores e seus animaizinhos.

### Pet Club

O projeto possui um aplicativo para Android que é responsável pelo desenvolvimento e manutenção de aplicativos que
interagem com a API do projeto.
